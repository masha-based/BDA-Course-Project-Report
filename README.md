# BDA-Course-Project-Report
Repository created for Project Report on Basics of Data Analysis Course // Data Science Master Programme, HSE Moscow

By Arman Malkhasyan and Maria Karnyushina

Source: [Kaggle](https://www.kaggle.com/datasets/anassarfraz13/student-success-factors-and-insights?resource=download)

Dataset: **Student Success: Factors & Insights**


We conducted the following research on the selected dataset:

1. 2D Regression and correlation analysis
2. PCA: Hidden factor & data visualization
3. K-means clustering  and cluster Interpretation
4. Contingency table analysis
5. Bootstrap for comparing averages
